# AppleApp（iOS 相册与视频工具）

**说明文档与功能相关更新日期**：2026年5月12日

**一句话**：这是一款在 iPhone 上使用的 **相册整理 + 简单视频制作 + 多路视频播放** 工具，适合处理「截图和保存图太多不好删」「想快速做带配乐的图片视频」「需要同时看多路画面」等日常需求；在 **「相机」** 里还可以用 **自定义相机** 拍照、录像并实时换画面效果。

---

## 这个 App 能帮你做什么

主要包括下面几类：

| 类型 | 说明 |
|------|------|
| **整理相册** | 把「不是手机相机拍的」照片/视频单独列出来（例如截图、聊天里保存的图、其他 App 存的图），方便预览和批量删除，不用在整本相册里翻来翻去。 |
| **做简单视频** | 给一段视频换一条音轨，或者把多张图配上音乐做成一段小视频（类似简易电子相册或短视频素材）。 |
| **多画面看视频** | 在同一个屏幕上同时播放多路视频（例如 4 格、6 格），点哪一格就听哪一路的声音；也支持看网络视频、直播和相册里的本地视频。 |
| **自定义相机** | 在 App 底部 **「相机」** 里 **拍照、录像**，取景时就能切换猫脸贴纸、热感、工笔等多种画面效果，录像可带上当时开着的效果；具体操作见下文 **第 9 条**。 |

除了这些实用功能，还有一些和 **画面效果、播放** 相关的进阶能力，适合对视频播放、多画面显示感兴趣的用户尝试。

## 功能列表（逐项说明）

### 1. 非相机照片与视频

**解决什么问题**：相册里混着「自己拍的」和「别处存的」（截图、保存的图片等），想集中处理后者。

**能做什么**：

- 列出非系统相机来源的图片和视频  
- 预览  
- 支持批量删除或一键全部删除  

---

### 2. Mac 端图片导入

**解决什么问题**：在 Mac 上有一批图片，希望批量弄到手机里统一管理。

**能做什么**：

- 与 Mac 端配套服务配合，从 Mac **批量导入图片到手机**  
- 适合素材整理、批量保存等场景  

---

### 3. 音轨合并（视频 + 音频）

**解决什么问题**：有一段视频，想换掉原来的声音，或配上别的音乐/人声。

**能做什么**：

- 选一个视频作为画面  
- 再选一个音频文件，或从另一个视频里 **只取声音**  
- 导出为新视频  
- 可按 **以画面长度为准** 或 **以声音长度为准** 对齐  

**常见用途**：换背景音乐、简单 vlog 配音等。

---

### 4. 图生视频（图片 + 音乐做成视频）

**解决什么问题**：手里有多张图和一段音乐，想自动做成一段可分享的视频。

**能做什么**：

- 选一张或多张图片  
- 选音频，或从别的视频里只取音轨  
- 设置每张图显示多久、切换效果  
- 按 **图片总时长** 或 **音频时长** 决定成片长度  
- 可保存到 App 文稿或系统相册，可改文件名，可预览  

**常见用途**：音乐相册、活动回顾、短视频素材等。

---

### 5. 单路 Metal 播放器

**是什么**：用苹果系统的 **Metal** 图形能力做的一路视频播放，画面可以做一些特殊显示效果。

**当前**：例如单色显示等效果。  

**后续**：计划增加滤镜、调色、模糊、锐化等。

---

### 6. FFmpeg 播放器

**是什么**：内置 **FFmpeg** 播放能力，用来在手机上流畅观看各类视频。

**支持**：HTTPS 网络视频、RTMP 直播流、相册里的本地视频等。

---

### 7. 四路 Metal 播放器

**能做什么**：屏幕上 **同时放 4 个视频**；点哪个画面，就播放哪一路的声音。

**适合场景**：多画面一起看、多段素材对比、需要同时预览多路内容时。

---

### 8. 六路 Metal 播放器

**能做什么**：与四路类似，升级为 **6 个画面同时播放**，同样支持点击切换听哪一路声音。

**适合场景**：分屏更多、对比或展示内容更丰富时。

---

### 9. 自定义相机（App 底部「相机」标签）

**解决什么问题**：想用 **自带界面** 拍照、录像，并希望在 **取景时就能直接看到** 各种好玩或艺术化的画面效果，而不必拍完再用别的 App 慢慢调色。

**能做什么**：

- 切换 **拍照 / 录像**，用快门拍照，或开始、停止录像（录像时一般会把 **麦克风环境声** 一起录进去，具体以手机系统权限为准）。  
- 底部 **横向滑动** 选择不同 **特效分类**：例如 **单色**（红 / 绿 / 蓝 / 灰，在弹出列表里选）、**蜡笔**、**猫脸**（对着人时脸上会出现小猫贴纸）、**热感**、**工笔**、**油画**、**水彩**、**壁画**，以及几种偏 **卡通、线描、重彩** 的风格。  
- **同一种特效再点一次**，通常可以 **关掉** 该效果；**单色** 想恢复自然颜色时，请再点「单色」并在列表里选 **「正常」**（点「取消」不会改变当前画面）。  
- **录像** 时，成片里会带上你 **当时打开的效果**，方便直接保存或分享。  

**常见用途**：自拍玩特效、给孩子录一段带贴纸的短片、旅行随手拍想换一种「画风」。

## 用什么做的（简要）

- **运行环境**：iPhone，系统为 iOS。  
- **界面与逻辑**：Swift、SwiftUI 等苹果常用开发方式。  
- **相册与文件**：读写系统相册、本地文件，完成整理与导出。  
- **画面与播放**：Metal 负责多路画面；FFmpeg 负责多种格式的网络与本地播放。  
- **自定义相机**：同样用系统能力实时处理画面（你不需要懂技术名词也能用）。  
- **与电脑协作**：通过配套能力从 Mac 批量导入图片（见上文「Mac 端图片导入」）。

## 开发与实现补充（偏技术，可选阅读）

下面这段是给 **想对照代码、自己编译或 fork** 的读者看的；**不影响**上面「这个 App 能帮你做什么」的阅读。其中对「单路播放」等与上文不一致之处，以本节为准（上文面向不懂技术的描述保留原样）。

### 工程与模块

- 主应用目标目录：`Photo related/`，Xcode 工程：`Photo related.xcodeproj`。  
- 主界面为 **Tab**：照片、视频播放、相机（SwiftUI `TabView`）。

### 单路播放与自定义相机（与上文第 5 条的关系）

- 单路播放与 **自定义相机** 共用同一套 **`PreviewEffectPipeline`**：在 Metal 表面上呈现，中间用 **Core Image** 做热感、工笔、油画、水彩、壁画、蜡笔等，用 **Metal 着色器** 做 Crayon / Sketch / 卡通3 等；**猫脸** 使用 **Vision** 做人脸框检测后贴纸。  
- 底部可选多种风格（与相机侧观感对齐）；再次点击当前已选特效可恢复「正常」。  
- FFmpeg 学习/预览等仍可使用 **NV12 → Metal `fragment_main`** 的轻量路径（主要为简单单色），与单路「相机同款特效」路径不同。

### 其他技术点（摘要）

| 技术 | 用途 |
|------|------|
| Metal / MetalKit | 多路 NV12 合成、Magic 系列 BGRA 处理、最终像素呈现 |
| Core Image / Vision | 风格化滤镜、猫脸贴纸与人脸检测 |
| AVFoundation | 相机、系统播放、音视频导出 |
| FFmpeg（`ThirdParty/ffmpeg`） | 解封装与解码、网络流等 |

- 全工程共享默认 Metal 库：**`MetalDefaultLibraryCache`**，App 启动 **`warmUp()`** 后台预热，减轻首次进入相关页面的卡顿。

### 本地构建（开发者）

- 建议使用与工程 **Deployment Target** 一致的 Xcode（当前为 **iOS 26.4+**）。  
- FFmpeg 头文件在 `ThirdParty/ffmpeg/include/`；克隆到干净环境后，需在 Xcode 中配置与你本机一致的 **FFmpeg 静态库 / xcframework** 搜索路径与链接。

```bash
git clone <你的仓库 URL>
cd AppleApp
open "Photo related.xcodeproj"
```

在 Xcode 中选择目标 **Photo related**，连接真机或模拟器后 **Run (⌘R)** 编译运行。
